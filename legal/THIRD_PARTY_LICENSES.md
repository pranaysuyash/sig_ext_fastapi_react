## License Summaries

### MIT License
Permissive license allowing commercial use, modification, distribution, and private use. Requires preservation of copyright and license notices.

### Apache License 2.0
Permissive license similar to MIT but with explicit patent grant. Allows commercial use, modification, distribution, and private use.

### BSD License (2-Clause and 3-Clause)
Permissive licenses allowing redistribution with minimal restrictions. Requires preservation of copyright notices.

### LGPL v3 (Lesser General Public License)
Allows use in proprietary software as long as the LGPL library itself remains open source and modifications to it are shared.

### AGPL v3 (Affero General Public License)
Similar to GPL but requires source code disclosure for network use. PyMuPDF is used under AGPL terms.

### MPL 2.0 (Mozilla Public License)
File-level copyleft license allowing combination with proprietary code. Modified MPL files must remain open source.

### HPND License (Historical Permission Notice and Disclaimer)
Very permissive license similar to MIT/BSD.
