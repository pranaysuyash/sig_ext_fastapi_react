## ðŸ”§ Customization Guide

### Easy Changes

**Colors:**
Edit in [css/style.css](css/style.css) lines 8-14:
```css
:root {
    --blue: #3b82f6;   /* Change to your brand color */
    --orange: #f59e0b; /* Change CTA color */
}
```

**Copy:**
All text is in [index.html](index.html) with clear section comments.

**Animations:**
Adjust speeds in [css/animations.css](css/animations.css):
```css
.animate-fade-up {
    animation: fadeUp 0.6s ease-out; /* Change 0.6s */
}
```

### Advanced Changes

**Add New Section:**
1. Copy an existing section structure from HTML
2. Style it using existing classes or add new ones
3. Add scroll animation class: `animate-on-scroll`

**New Animation:**
1. Define keyframes in animations.css
2. Create animation class
3. Apply to elements

**Integration:**
- Payment: Add Stripe/PayPal in CTA click handlers
- Analytics: Add code before `</body>`
- Email: Set up form handling in main.js
