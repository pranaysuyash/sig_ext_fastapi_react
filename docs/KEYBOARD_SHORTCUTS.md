# Keyboard Shortcuts

Complete list of keyboard shortcuts for the Signature Extractor app.

---

## File Operations

| Shortcut | Action | Description |
|----------|--------|-------------|
| `⌘O` or `Ctrl+O` | Open Image | Open and upload an image file |
| `⌘E` or `Ctrl+E` | Export | Export the extracted signature |
| `⌘L` or `Ctrl+L` | Save to Library | Save current result to signature library |
| `⌘Shift+O` or `Ctrl+Shift+O` | Open PDF | Open a PDF document (PDF tab) |
| `⌘Shift+S` or `Ctrl+Shift+S` | Save PDF | Save signed PDF document (PDF tab) |

---

## Editing & Processing

| Shortcut | Action | Description |
|----------|--------|-------------|
| `⌘C` or `Ctrl+C` | Copy | Copy extracted signature to clipboard |
| `⌘Shift+V` or `Ctrl+Shift+V` | Paste Signature | Paste signature from clipboard (PDF tab) |
| `⌘[` or `Ctrl+[` | Rotate CCW | Rotate counter-clockwise |
| `⌘]` or `Ctrl+]` | Rotate CW | Rotate clockwise |

---

## View & Navigation

| Shortcut | Action | Description |
|----------|--------|-------------|
| `⌘+` or `Ctrl++` | Zoom In | Increase zoom level of active pane |
| `⌘-` or `Ctrl+-` | Zoom Out | Decrease zoom level of active pane |
| `⌘0` or `Ctrl+0` | Reset Viewport | Reset zoom and pan to default |
| `⌘1` or `Ctrl+1` | Fit to View | Fit image to viewport |

---

## Workflow

| Shortcut | Action | Description |
|----------|--------|-------------|
| `P` | Preview | Process selection and show preview |
| `T` | Toggle Selection Mode | Switch between Select and Pan modes |
| `A` | Toggle Auto Threshold | Enable/disable automatic threshold |
| `Esc` | Cancel/Deselect | Clear current selection or close dialogs |

---

## Tips

### macOS-Specific:
- Use `⌘` (Command) instead of `Ctrl` on macOS
- Native file dialogs support drag-and-drop
- Use trackpad gestures for zoom/pan (if enabled)

### Workflow Shortcuts:
1. **Quick Extract**: `⌘O` → Select area → `P` → `⌘C`
2. **Save to Library**: `⌘O` → Select area → `P` → `⌘L`
3. **Sign PDF**: `⌘Shift+O` → Select signature → Click → `⌘Shift+S`

### Customization:
Shortcuts follow standard macOS/Qt conventions and cannot be customized at this time.
